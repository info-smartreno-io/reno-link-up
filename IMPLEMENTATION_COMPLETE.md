## 🤖 Site Health Copilot

The crown jewel of the system - an AI assistant that monitors and helps manage:

**Capabilities:**
- Answers questions about SEO, performance, errors, and content
- References all generated reports for context
- Provides actionable recommendations
- Helps diagnose issues and suggest fixes
- Analyzes trends across all monitoring systems

**Example Questions:**
- "Why did my performance score drop?"
- "What broken links need fixing?"
- "Show me critical errors from this week"
- "Which SEO pages need updates?"
- "What content should I create next?"

**Access:** `/admin/ai-dashboard` → AI Copilot tab

The copilot uses Lovable AI (Gemini 2.5 Flash) and has access to:
- Latest SEO reports and recommendations
- Performance audit results
- Broken link scans
- Error log analysis
- Content pipeline data

It provides instant, context-aware answers to help admins maintain site health without digging through multiple dashboards.
