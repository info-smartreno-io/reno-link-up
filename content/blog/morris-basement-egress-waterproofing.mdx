Finishing a **basement in Morris County** usually requires an egress window or door for sleeping areas and proper moisture control.

## Code Requirements for Bedrooms

<Callout type="error" title="Egress Window Required">
If you plan to use your finished basement as a bedroom in **Morris County**, IRC code requires an egress window with minimum opening dimensions (typically 5.7 sq ft, 24" height, 20" width).
</Callout>

Any bedroom (even in a basement) needs:
- **Window opening area:** Minimum 5.7 square feet
- **Clear opening height:** At least 24 inches
- **Clear opening width:** At least 20 inches
- **Sill height:** Maximum 44 inches from floor

## Waterproofing Checklist

- Layout plan
- Mechanical clearances
- Sump/french drain as needed
- Vapor barrier installation
- Insulation (closed‑cell foam recommended)
- Sound control between floors

<Callout type="info" title="Morris County Moisture">
Towns like **Parsippany, Randolph, and Morristown** have varying water table levels. A professional assessment can determine if you need a sump pump or french drain system.
</Callout>

## Common Basement Layouts

1. **Family Room + Bath** – Most popular configuration
2. **Guest Suite** – Requires egress window for bedroom
3. **Home Office + Storage** – No egress needed
4. **Entertainment Zone** – Bar, TV area, game space

Proper waterproofing and egress add resale value and make your basement a true living space, not just storage.
</Callout>
