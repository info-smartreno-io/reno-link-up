This repository powers SmartReno.

SmartReno is a renovation platform that helps homeowners move from project intake to estimator scheduling, contractor bidding, and project coordination.

Priority systems:
1. Authentication
2. Homeowner intake
3. Estimator scheduling
4. Contractor bidding
5. Messaging
6. Payments
7. Walkthrough coordination
8. Admin workflows

Development must prioritize:
- production safety
- schema accuracy
- minimal regression risk
- secure role-based access
- reliable Supabase + Stripe flows
